# Homebrew Tap for Dimmerly

Install Dimmerly with Homebrew:

```bash
brew tap olujicz/dimmerly
brew install --cask dimmerly
```

Dimmerly is a native macOS menu bar app for display brightness, warmth,
contrast, presets, schedules, display sleep, and DDC/CI hardware controls.

Project: https://github.com/olujicz/Dimmerly
